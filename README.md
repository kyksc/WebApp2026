2. 작업 후 다시 올리는 순서 (중요!)
새 폴더를 만들고 코딩을 마쳤다면, 터미널(WebApp2026 폴더 안)에서 아래 명령어를 순서대로 입력하세요.

변경 사항 확인

PowerShell
git status
새로 만든 폴더가 빨간색으로 표시되는지 확인합니다.

스테이징 (올릴 준비)

PowerShell
git add .
add .을 하면 새로 만든 폴더와 파일들이 업로드 대기 상태가 됩니다.

커밋 (기록 남기기)

PowerShell
git commit -m "5주차 Props 실습 과제 제출"
이때 입력하는 메시지가 깃허브 히스토리에 남게 됩니다.

푸시 (서버 전송)

PowerShell
git push origin main
드디어 내 컴퓨터의 작업물이 온라인 깃허브 저장소로 합쳐집니다.
